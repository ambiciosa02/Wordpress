== Description ==

Awesome eCommerce Theme. The eCommerce Shop is the ultimate WordPress theme for WooCommerce. It is responsive, fast, customizable, SEO-Friendly and page builder ready WordPress theme. This theme is fully compatible with WooCommerce plugin and has additional theme options. This theme adds additional style and design to WooCommerce pages. It is Elementor page builder friendly theme so you can create and design eye-catching unique pages. Theme Features: One Click #Demo Import, #Responsive, #Fast to load, fully #Customizable, #WooCommerce Options, #Elementor page builder ready, Full Width #Template, #Blog Options, #Typography, Top Bar section, Footer #Widgets, #CTA, Header #Layouts, #Social Icons, Sidebar Cart, #Sticky Menu and much more. Let's create an awesome eCommerce website. #DEMO: http://demo.dithemes.com/di-ecommerce/
